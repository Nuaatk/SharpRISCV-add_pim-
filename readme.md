# SharpRISCV #
SharpRISCV is an implementation of RISC-V assembly in C#.

![Output Image](https://github.com/rizwan3d/SharpRISCV/blob/master/img.jpg?raw=true)

## Supported Instruction
1. [X] R Type
1. [X] U Type
1. [X] I Type
1. [X] B Type
1. [X] I Type
1. [X] S Type
1. [X] J Type
